**Automated F1 Race Data Analysis and Dashboard Integration**

Developed an end-to-end Formula 1 Race Data Analytics solution to automate the extraction, transformation, analysis, and visualization of Formula 1 racing data using modern Business Intelligence and cloud analytics tools. The project focused on building an automated data pipeline capable of collecting race statistics, processing datasets, and generating interactive dashboards for performance analysis and reporting.

**Project Overview :**

The project involved extracting Formula 1 race data from public APIs and integrating it into a scalable analytics workflow for monitoring driver performance, constructor standings, lap statistics, race outcomes, and season trends. The solution automated the complete analytics lifecycle from data ingestion to dashboard publishing and refresh scheduling.

**End-to-End Workflow :**

->Extracted Formula 1 race data from external APIs and online datasets

->Built automated data ingestion and transformation pipelines

->Performed data cleaning, preprocessing, and validation for accurate reporting

->Designed structured data models and relationships for analytical querying

->Created semantic models to improve dashboard performance and reporting efficiency

->Developed interactive dashboards and reports using Power BI Desktop

**Visualized:**

->Driver standings and rankings
->Constructor performance analysis
->Race results and lap statistics
->Fastest laps and qualifying insights
->Seasonal trends and performance comparisons
->Track-wise and team-wise analytics
->Published reports and dashboards to Power BI Service for cloud accessibility and sharing
->Configured automated scheduled refreshes for real-time data updates
->Implemented monitoring and workflow automation for reliable dashboard operations
->Enabled automated notifications using Power Automate for pipeline or refresh failures
